О проекте

Этот калькулятор создан на базе AI-модели для подбора освещения. Он учитывает требования нормативных документов (ГОСТ, СНиП, СанПиН) и помогает выбрать оптимальные светильники для разных помещений. Достаточно ввести параметры, и система рассчитает необходимое количество светильников, их тип и примерную стоимость.

Какие параметры вводятся?

  • Тип помещения — офис, кафе, склад, школа и др.
  • Площадь (м²) — обязательный параметр для расчёта.
  • Высота потолка (м) — используется, если помещение имеет потолок.
  • Бюджет (₽) — система подбирает решение, укладывающееся в заданный лимит.
